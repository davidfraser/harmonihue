Musical Notation

Modern musical notation has a rich and interesting history; it is generally based on the concept of scales, so that vertically-arranged notes on a staff (a set of horizontal grid lines) from the indicated key signature correspond to the desired scale.

Sample - Single Melody: Amazing Grace
{ \key d \major \time 3/4 \partial 4 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 fis''8 e'' d''2 b'4 a'2 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 e''8. fis''16 a''2 r4 r2 fis''8. a''16 a''2 fis''8[ e''32 d''16.] fis''2 fis''8 e'' d''2 b'4 a'2 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 e''4 d''2. r2 \bar "|." }

Simple Hue-based Coloring

Applying the hues to the heads of the notes gives the following:

Sample - Single Melody: Amazing Grace
\include "${chromaturn_ly_filename}" { \chromaNotesOn \key d \major \time 3/4 \partial 4 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 fis''8 e'' d''2 b'4 a'2 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 e''8. fis''16 a''2 r4 r2 fis''8. a''16 a''2 fis''8[ e''32 d''16.] fis''2 fis''8 e'' d''2 b'4 a'2 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 e''4 d''2. r2 \bar "|." }
Chromatic Scale and Circle of Fifths
\include "${chromaturn_ly_filename}" \relative { \chromaNotesOn \key c \major \time 3/4 c4 cis4 d4 ees4 e4 f4 fis4 g4 aes4 a4 bes4 b4 c2. \bar "||" c,4 g'4 d4 a'4 e4 b'4 fis4 cis'4 gis4 dis'4 ais4 f'4 c2. \bar "|." }

Hues and Rotation Indicators

Applying the hues to the heads of the notes and indicating the rotation around the torus with a line gives the following:

\include "${chromaturn_ly_filename}" { \chromaTurnOn \key d \major \time 3/4 \partial 4 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 fis''8 e'' d''2 b'4 a'2 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 e''8. fis''16 a''2 r4 r2 fis''8. a''16 a''2 fis''8[ e''32 d''16.] fis''2 fis''8 e'' d''2 b'4 a'2 a'8. d''16 d''2 fis''8[ e''32 d''16.] fis''2 e''4 d''2. r2 \bar "|." }
