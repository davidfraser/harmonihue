HarmoniHue

HarmoniHue is a system for relating musical pitch to visual phenomenen (especially colour). The aim is to find a correspondence between these different concepts that will function intuitively, especially in conveying the harmonic relationships between different pitches.

The following are sample scores generated using an adjusted lilypond with a special macro to include the harmonihue concepts in the notation.

  • Chopin Nocturne Op. 9, No. 1 in B flat minor (Lilypond source)
  • J.S. Bach Well-Tempered Klavier Fugue 2 (Lilypond source)
  • R. Schumann Romanze Op. 28 No. 2 (Lilypond source)
  • Debussy Arabesque (Lilypond source)
